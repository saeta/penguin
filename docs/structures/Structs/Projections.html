Projections

A Sequence whose elements are projections, through a lens of type BaseElementPart, of the elements of some Base sequence.

Projections is analogous to LazyMapSequence but doesn’t store a closure or a KeyPath, and when Base conforms to MutableCollection, so does Projections<Base, L>, as long as L.Focus is-a WritableKeyPath.

  • Element

    The type of each element of self.

  • Iterator

    Single-pass iteration interface and state for instances of Self.

    See more
  • makeIterator()

    Returns an iterator over the elements of this sequence.

  • underestimatedCount

    A value <= self.count.

  • Index

    A position in an instance of Self.

  • Indices

  • startIndex

    The position of the first element, or endIndex if self.isEmpty.

  • endIndex

    The position immediately after the last element.

  • indices

    The indices of all elements, in order.

  • index(after:)

    The position following x.

  • formIndex(after:)

    Replaces x with its successor

  • subscript(_:)

    Accesses the element at i.

  • isEmpty

    True iff self contains no elements.

  • count

    The number of elements in self.

    Complexity

    O(1) if Base conforms to RandomAccessCollection; otherwise, O(N) where N is the number of elements.
  • index(_:offsetBy:)

    Returns an index that is the specified distance from the given index.

    Complexity

    O(1) if Base conforms to RandomAccessCollection; otherwise, O(distance).
  • index(_:offsetBy:limitedBy:)

    Returns i offset forward by distance, unless that distance is beyond a given limiting index, in which case nil is returned.

    Complexity

    O(1) if Base conforms to RandomAccessCollection; otherwise, O(distance).
  • distance(from:to:)

    Returns the number of positions between start to end.

    Complexity

    O(1) if Base conforms to RandomAccessCollection; otherwise, worst case O(count).
  • index(before:)

    Returns the position immediately before i.

  • formIndex(before:)

    Replaces the value of i with its predecessor.
